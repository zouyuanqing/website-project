时间显示测试

时间显示修复说明

系统已修复时间显示问题，现在所有时间都会自动从UTC时间转换为北京时间（UTC+8）显示。

时间格式示例：
格式说明 过滤器 显示效果
完整时间 |local_time {{ now() | local_time }}
仅日期 |local_date {{ now() | local_date }}
简短时间 |local_time_short {{ now() | local_time_short }}
自定义格式 |local_time('%Y年%m月%d日 %H:%M') {{ now() | local_time('%Y年%m月%d日 %H:%M') }}
使用方法：

在模板中使用过滤器：

# 显示完整时间（年-月-日 时:分:秒）
{{ user.created_at | local_time }}

# 仅显示日期
{{ form.created_at | local_date }}

# 显示简短时间（月-日 时:分）
{{ submission.submitted_at | local_time_short }}

# 自定义格式
{{ payment.paid_at | local_time('%Y年%m月%d日') }}
修复范围
  • 用户注册时间
  • 表单创建和更新时间
  • 提交记录时间
  • 支付订单时间
  • 文件上传时间
  • 所有其他系统时间戳
